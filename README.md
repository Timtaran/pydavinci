# pydavinci

Not on PyPy yet. If you want to test it out, clone the repository
```bash
git clone https://github.com/pedrolabonia/pydavinci.git
cd pydavinci
pip install -e .
```

Then, use by importing it:

```python
from pydavinci import Resolve

resolve = Resolve()
```

Requires Python 3.6.*

WIP Docs are at /docs.
